## 9x9 Sudoku Solver in C
This source code solves only 9x9 sudoku.
There is no error code for now, so please be careful to fill in correctly.

## Input Format

- First, you need to complete the line then press enter.
- Don't forget the enter 0(zero) if it is unassigned.

###### Example 1:
    4 9 0 0 7 0 8 0 6
    0 6 0 0 9 5 0 1 0
    0 0 1 3 0 0 0 0 9
    0 5 0 2 0 0 0 0 4
    6 8 0 0 3 4 0 7 5
    1 0 0 0 8 0 0 6 0
    3 0 0 0 0 1 6 0 0
    0 2 0 6 5 0 0 9 0
    5 0 6 0 4 0 0 2 3
###### Example 2:
    4 0 0 0 0 2 5 0 0
    0 0 2 0 3 0 6 0 0
    0 0 0 0 0 9 0 0 7
    0 0 0 1 5 0 4 0 0
    0 0 0 0 0 6 0 1 0
    0 9 0 0 0 0 0 0 3
    0 0 3 0 1 0 0 0 2
    0 6 5 0 0 0 0 0 0
    0 0 0 7 0 3 0 0 0
